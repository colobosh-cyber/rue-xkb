# Rusyn RUE Keyboard (Windows)

## English

Rusyn RUE is a practical keyboard layout for Windows, designed for everyday Rusyn writing.

It is based on the Ukrainian layout but adapted for real usage, with additional letters and useful symbols.

---

### Installation

1. Open the `windows` folder.
2. Run `setup.exe`.
3. Complete the installation.
4. Open **Settings → Time & Language → Language & Region**.
5. Select **Ukrainian** → **Keyboard**.
6. Add and select **Rusyn RUE**.

---

### Important Note

Windows will display the layout under:

**UKR (Ukrainian)**

This is normal and expected.
The layout itself is **Rusyn RUE**.

---

### Features

* Rusyn-friendly layout based on Ukrainian

* Additional letters:

  * ы
  * ӱ / Ӱ
  * ґ / Ґ
  * ї / Ї
  * є / Є

* Useful symbols:

  * № € ©
  * « »
  * < >
  * — ±
  * → ← × ÷
  * [ ] { } \ |
  * ' " ` ~
  * °

* Accent (stress mark):

  * Located on **AltGr + OEM_3**
  * Matches Ubuntu layout position
  * Works as a dead key

---

### Notes

* The layout is designed for practical everyday typing.
* Some Unicode combinations (like accented Cyrillic letters) may depend on the application and font.
* All core symbols and letters work reliably in Windows.

---

All files in the `windows` folder are required for installation.
Do not remove or move any files before running setup.exe.

---

### Version

RUE Keyboard 1.0

## Українська

Rusyn RUE — це практична клавіатурна розкладка для Windows, створена для щоденного використання русинської мови.

Вона базується на українській розкладці, але адаптована для реального використання та доповнена необхідними літерами і символами.

---

### Встановлення

1. Відкрийте папку `windows`.
2. Запустіть `setup.exe`.
3. Завершіть встановлення.
4. Відкрийте **Налаштування → Час і мова → Мова та регіон**.
5. Оберіть **Українська** → **Клавіатура**.
6. Додайте та виберіть **Rusyn RUE**.

---

### Важливо

Windows буде відображати розкладку як:

**UKR (Українська)**

Це нормально.
Сама розкладка — **Rusyn RUE**.

---

### Особливості

* Русинська розкладка на базі української

* Додаткові літери:

  * ы
  * ӱ / Ӱ
  * ґ / Ґ
  * ї / Ї
  * є / Є

* Корисні символи:

  * № € ©
  * « »
  * < >
  * — ±
  * → ← × ÷
  * [ ] { } \ |
  * ' " ` ~
  * °

* Наголос:

  * знаходиться на **AltGr + OEM_3**
  * відповідає розкладці Ubuntu
  * працює як dead key

---

### Примітки

* Розкладка створена для практичного щоденного використання.
* Деякі Unicode-комбінації (наприклад наголоси) можуть залежати від програми або шрифту.
* Основні символи та літери працюють стабільно у Windows.
